---

File Management

Ctrl + N: Create a new document.

Ctrl + O: Open an existing document.

Ctrl + S: Save the current document.

F12: Open the "Save As" dialog box.

Ctrl + W: Close the current document.

Ctrl + P: Print the document.

Alt + F: Open the File menu.



---

Text Selection

Shift + Arrow Keys: Select text character by character or line by line.

Ctrl + Shift + Arrow Keys: Select word by word or paragraph by paragraph.

Ctrl + A: Select the entire document.

Shift + End: Select from the cursor to the end of the line.

Shift + Home: Select from the cursor to the beginning of the line.



---

Text Formatting

Ctrl + B: Bold text.

Ctrl + I: Italicize text.

Ctrl + U: Underline text.

Ctrl + Shift + W: Apply a single underline to words only (not spaces).

Ctrl + Shift + D: Double underline text.

Ctrl + Shift + A: Change text to all caps.

Ctrl + =: Apply subscript.

Ctrl + Shift + =: Apply superscript.

Ctrl + Space: Remove manual character formatting.



---

Paragraph Formatting

Ctrl + E: Center align the paragraph.

Ctrl + L: Left-align the paragraph.

Ctrl + R: Right-align the paragraph.

Ctrl + J: Justify the paragraph.

Ctrl + M: Increase paragraph indent.

Ctrl + Shift + M: Decrease paragraph indent.

Ctrl + T: Create a hanging indent.

Ctrl + Shift + T: Remove a hanging indent.

Ctrl + Q: Remove paragraph formatting.



---

Navigating the Document

Ctrl + Arrow Keys: Move by word or paragraph.

Ctrl + Home: Move to the beginning of the document.

Ctrl + End: Move to the end of the document.

Ctrl + Page Up: Move to the top of the previous page.

Ctrl + Page Down: Move to the top of the next page.



---

Working with Tables

Tab: Move to the next cell in a table.

Shift + Tab: Move to the previous cell in a table.

Alt + Home: Move to the first cell in the row.

Alt + End: Move to the last cell in the row.

Alt + Page Up: Move to the first cell in the column.

Alt + Page Down: Move to the last cell in the column.



---

Special Shortcuts

Ctrl + Shift + C: Copy formatting.

Ctrl + Shift + V: Paste formatting.

Ctrl + K: Insert a hyperlink.

Ctrl + F: Open the Find pane.

Ctrl + H: Open the Replace pane.

Ctrl + G: Go to a specific page, section, or bookmark.

Alt + Shift + D: Insert the current date.

Alt + Shift + T: Insert the current time.



---

Ribbon Navigation

Alt: Show Key Tips for ribbon commands.

Alt + H: Go to the Home tab.

Alt + N: Go to the Insert tab.

Alt + P: Go to the Page Layout tab.

Alt + R: Go to the Review tab.

Alt + W: Go to the View tab.



---

Working with Graphics

Alt + Click: Select an object behind text.

Ctrl + Click: Select multiple objects.

Shift + Arrow Keys: Resize graphics proportionally.
